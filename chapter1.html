Introduction To C Language

What is C Language?

C is a general-purpose programming language developed by Dennis Ritchie in 1972 at Bell Labs. It is one of the most powerful programming languages and is considered the foundation of many modern languages such as C++, Java and C#.

Why Learn C?

  • C is fast and efficient.
  • It teaches programming fundamentals.
  • Used in Operating Systems.
  • Used in Embedded Systems.
  • Helps understand memory management.
  • Foundation of many programming languages.

Features of C

  • Simple Language
  • Portable
  • Structured Programming
  • Fast Execution
  • Rich Library Functions
  • Middle Level Language

First C Program

#include <stdio.h> int main() { printf("Hello World"); return 0; }

Program Explanation

  • #include <stdio.h> → Standard Input Output Library
  • main() → Starting point of every C program
  • printf() → Used to display output
  • return 0 → Program executed successfully

Important Interview Question

Q: Who developed C Language?

Answer: Dennis Ritchie in 1972 at Bell Laboratories.
