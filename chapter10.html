Chapter 10 - Pointers in C

What is a Pointer?

A Pointer is a variable that stores the memory address of another variable. Instead of storing a value directly, a pointer stores the location where the value exists.

💡 Pointer = Address Holder

Why Do We Use Pointers?

✔ Efficient Memory Usage

✔ Dynamic Memory Allocation

✔ Fast Array Processing

✔ Pass Data to Functions Efficiently

✔ Used in Data Structures

Memory Concept

Every variable is stored at a memory address.

int age = 20;
Value = 20 Address = 1000 (Example)

Address Operator (&)

The & operator returns the address of a variable.

int age = 20; printf("%p",&age);

Pointer Declaration

int *ptr;

ptr is a pointer that can store the address of an integer variable.

Pointer Initialization

int age = 20; int *ptr = &age;

ptr now stores the address of age.

Pointer Diagram

age = 20 Address of age = 1000 ptr = 1000
ptr ↓ age = 20

Dereference Operator (*)

The * operator is used to access the value stored at an address.

int age = 20; int *ptr = &age; printf("%d",*ptr);
Output: 20

Complete Example

#include int main() { int num = 50; int *ptr = # printf("Value = %d\n",num); printf("Address = %p\n",&num); printf("Pointer Value = %d",*ptr); return 0; }

Changing Value Through Pointer

int num = 10; int *ptr = # *ptr = 100; printf("%d",num);
Output: 100

The original variable changes because the pointer accesses the same memory location.

Pointer and Variable Relationship

num = 10 ptr = address of num *ptr = value stored in num

Real Life Example

Imagine a house.

House = Variable House Address = Pointer

If you know the address, you can reach the house. Similarly, if a pointer knows the address of a variable, it can access its value.

Pointer Arithmetic

Pointers can move through memory locations.

int arr[5]={10,20,30,40,50}; int *ptr = arr; printf("%d\n",*ptr); ptr++; printf("%d",*ptr);
Output: 10 20
💡 ptr++ moves to the next memory location of the same data type.

Pointers and Arrays

int arr[3]={10,20,30}; int *ptr = arr; printf("%d",*(ptr+1));
Output: 20

Array name itself stores the address of the first element.

Arrays vs Pointers

arr = Address of First Element ptr = Variable Holding Address
int arr[5]; int *ptr = arr;

Traversing Array Using Pointer

int arr[5]={10,20,30,40,50}; int *ptr = arr; for(int i=0;i<5;i++) { printf("%d ",*(ptr+i)); }
10 20 30 40 50

Call By Value

void change(int x) { x = 100; }

Original value does not change.

Call By Reference

void change(int *x) { *x = 100; } int main() { int num = 10; change(&num); printf("%d",num); }
Output: 100

Pointer allows the function to modify the original variable.

NULL Pointer

int *ptr = NULL;

NULL means the pointer is not pointing to any valid memory location.

Wild Pointer

int *ptr;

A pointer declared but not initialized is called a Wild Pointer.

⚠ Wild pointers may contain garbage addresses.

Dangling Pointer

A pointer pointing to deleted or invalid memory is called a Dangling Pointer.

int *ptr; { int x = 10; ptr = &x; }

After x goes out of scope, ptr becomes dangling.

Pointer to Pointer

int num = 10; int *ptr = # int **pptr = &ptr; printf("%d",**pptr);
Output: 10
