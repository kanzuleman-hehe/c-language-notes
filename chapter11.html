What is a Structure?

A Structure is a user-defined data type that allows us to store different types of data under one name.

💡 Structure can store int, float, char and other data types together.

Why Use Structures?

✔ Store Student Records

✔ Store Employee Information

✔ Store Product Details

✔ Organize Data Efficiently

Structure Syntax

struct Student { char name[50]; int age; float cgpa; };

Creating Structure Variables

struct Student s1;

Accessing Structure Members

s1.age = 20; printf("%d",s1.age);

Complete Structure Example

#include struct Student { char name[50]; int age; float cgpa; }; int main() { struct Student s1; s1.age=20; s1.cgpa=3.8; printf("%d\n",s1.age); printf("%.2f",s1.cgpa); return 0; }

Input Structure Data

struct Student s1; scanf("%s",s1.name); scanf("%d",&s1.age); scanf("%f",&s1.cgpa);

Display Structure Data

printf("%s",s1.name); printf("%d",s1.age); printf("%.2f",s1.cgpa);

Array of Structures

Used when storing multiple records.

struct Student s[100];

Example - Multiple Students

struct Student s[3]; for(int i=0;i<3;i++) { scanf("%s",s[i].name); }

Nested Structures

struct Address { char city[30]; }; struct Student { char name[30]; struct Address add; };

Access Nested Structure

printf("%s",s1.add.city);

Structure Pointer

A pointer can also point to a structure.

struct Student { char name[30]; int age; }; struct Student s1; struct Student *ptr = &s1;

Arrow Operator (->)

Used to access structure members through pointers.

ptr->age = 20; printf("%d",ptr->age);

What is File Handling?

File Handling allows data to be stored permanently in files. Without files, data is lost when the program closes.

💡 File Handling is used in Student Management Systems, Banking Systems and Database Applications.

File Pointer

FILE *fp;

FILE is a predefined structure used to manage files.

Opening a File - fopen()

FILE *fp; fp = fopen("student.txt","w");

"w" means Write Mode.

Closing a File - fclose()

fclose(fp);

Always close files after use.

Writing to File - fprintf()

FILE *fp; fp = fopen("data.txt","w"); fprintf(fp,"Hello World"); fclose(fp);

Reading from File - fscanf()

FILE *fp; char name[50]; fp = fopen("data.txt","r"); fscanf(fp,"%s",name); printf("%s",name); fclose(fp);

File Modes

r = Read w = Write a = Append r+ = Read + Write w+ = Write + Read a+ = Append + Read

Append Mode Example

FILE *fp; fp = fopen("data.txt","a"); fprintf(fp,"New Record"); fclose(fp);

Append mode adds data at the end of the file.

Binary File Writing - fwrite()

fwrite(&s1, sizeof(s1), 1, fp);

Used for writing structures directly into binary files.

Binary File Reading - fread()

fread(&s1, sizeof(s1), 1, fp);

Student Record Example

struct Student { char name[50]; float cgpa; }; FILE *fp; fp = fopen("students.txt","w"); fprintf(fp, "Ahmed 3.90"); fclose(fp);

MCQs

1. Which function opens a file?

A) open()
B) fopen()
C) fileopen()
D) createfile()

✅ Answer: B


2. Which function closes a file?

A) fclose()
B) fileclose()
C) close()

3. Which mode is used for appending data?

A) r
B) w

4. Which operator accesses structure members using a pointer?
