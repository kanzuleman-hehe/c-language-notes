What is a Variable?

A variable is a named memory location used to store data. Example: Age = 20 Here Age is a variable and 20 is its value.

Rules for Naming Variables

  • Must start with a letter or underscore (_)
  • Cannot start with a number
  • No spaces allowed
  • No special characters except underscore
  • Keywords cannot be used as variable names

Data Types in C

Data Type Used For Example
int Integer Values 10
float Decimal Values 3.14
char Single Character A
double Large Decimal Values 12.345678

Declaring Variables

int age = 20; float cgpa = 3.85; char grade = 'A';

Input From User

#include int main() { int age; printf("Enter Age: "); scanf("%d",&age); printf("Age = %d",age); return 0; }

Output

Enter Age: 20 Age = 20

Format Specifiers

Specifier Meaning
%d Integer
%f Float
%c Character
%lf Double
%s String

Important Interview Questions

Q1: What is a Variable?

Q2: Which data type stores decimal numbers?

Answer: float or double
