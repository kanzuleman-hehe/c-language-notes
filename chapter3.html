Operators in C Language


What is an Operator?

An Operator is a symbol that performs an operation on operands (values or variables). Example: 5 + 3 Here: + = Operator 5 and 3 = Operands

Types of Operators in C

Operator Type Example
Arithmetic + - * / %
Relational == != > < >= <=
Logical && || !
Assignment = += -= *=
Increment/Decrement ++ --

Arithmetic Operators

Operator Description
+ Addition
- Subtraction
* Multiplication
/ Division
% Modulus
#include<stdio.h> int main() { int a=10,b=5; printf("Addition = %d\n",a+b); printf("Subtraction = %d\n",a-b); printf("Multiplication = %d\n",a*b); printf("Division = %d\n",a/b); return 0; }

Relational Operators

Relational Operators compare two values. Result is either: 1 = True 0 = False

#include<stdio.h> int main() { int a=10,b=20; printf("%d",a

Output: 1 Because 10 is less than 20.

Assignment Operators

int a=10; a+=5; printf("%d",a);

Output = 15

Increment & Decrement

int x=5; x++; printf("%d",x);

Output = 6

🚀 Advanced Knowledge Corner

Operator Precedence decides which operator executes first. Example:

int result = 10 + 5 * 2; printf("%d",result);

Output = 20 Because multiplication (*) has higher precedence than addition (+). Calculation: 5 × 2 = 10 10 + 10 = 20

Common Mistakes

  • Using = instead of ==
  • Forgetting operator precedence
  • Division by zero
  • Using wrong logical operators
