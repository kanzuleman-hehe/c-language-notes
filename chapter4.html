What is Input & Output?

Input means taking data from the user. Output means displaying data to the screen. In C Language: printf() → Output Function scanf() → Input Function

printf() Function

printf() is used to display output on the screen.

#include<stdio.h> int main() { printf("Welcome To C Language"); return 0; }

scanf() Function

scanf() is used to take input from the user.

#include<stdio.h> int main() { int age; scanf("%d",&age); printf("%d",age); return 0; }

Format Specifiers

Specifier Meaning
%d Integer
%f Float
%c Character
%s String
%lf Double

Example Program

#include<stdio.h> int main() { char name[20]; int age; printf("Enter Name: "); scanf("%s",name); printf("Enter Age: "); scanf("%d",&age); printf("Name = %s\\n",name); printf("Age = %d",age); return 0; }

🚀 Advanced Knowledge Corner

scanf() stores data directly into memory. The symbol & is called Address Operator. Example:

scanf("%d",&age);

&age means: "Store the entered value at the memory address of age." This concept becomes very important when learning pointers.

Common Mistakes

  • Forgetting & in scanf()
  • Using wrong format specifier
  • Using %d for float values
  • Misspelling printf() or scanf()

Interview Questions

Q1: Difference between printf() and scanf()?

printf() → Displays output scanf() → Takes input

Q2: Why is & used in scanf()?

Because scanf needs the variable's memory address.

MCQs

1. Which function is used for output?

2. Which symbol is used before variable in scanf()?
