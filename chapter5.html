What are Conditional Statements?

Conditional Statements allow a program to make decisions based on conditions. If the condition is true, one block executes. If false, another block executes.

1. if Statement

if(age >= 18) { printf("Eligible"); }
Output: Eligible

2. if-else Statement

if(age >= 18) { printf("Adult"); } else { printf("Minor"); }

3. else-if Ladder

if(marks >= 80) { printf("A Grade"); } else if(marks >= 70) { printf("B Grade"); } else { printf("C Grade"); }

4. Nested if

if(age >= 18) { if(citizen == 1) { printf("Eligible To Vote"); } }

Nested if means an if statement inside another if statement.

5. switch Statement

switch(choice) { case 1: printf("Tea"); break; case 2: printf("Coffee"); break; default: printf("Invalid Choice"); }

🚀 Advanced Knowledge Corner

Many beginners write:

if(x = 5)
⚠ Wrong! This assigns 5 to x.
if(x == 5)
✅ Correct! This compares x with 5.

Real World Example

if(balance >= 500) { printf("Transaction Approved"); } else { printf("Insufficient Balance"); }

Banks, ATMs and online payment systems use conditions like this every day.

Interview Questions

Q1: Difference between if and switch?

Answer: if can check ranges and complex conditions. switch checks exact values only.

Q2: What is Nested if?

An if statement inside another if statement.
