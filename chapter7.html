What is a Function?

A Function is a block of code that performs a specific task. Instead of writing the same code repeatedly, we can write it once inside a function and call it whenever needed.

💡 Functions make programs shorter, cleaner and easier to maintain.

Why Do We Use Functions?

✔ Reduce Code Repetition

✔ Improve Readability

✔ Easier Debugging

✔ Better Program Structure

✔ Reusability

Real Life Example

Imagine a calculator application. Instead of writing addition code again and again, we create one function:

int add(int a,int b) { return a+b; }

Now we can use it anywhere.

Function Structure

return_type function_name(parameters) { // code }

Example:

int add(int a,int b) { return a+b; }

Function Declaration (Prototype)

A Function Prototype tells the compiler that a function exists.

int add(int,int);
💡 Prototype is usually written before main().

Function Definition

The actual code of the function.

int add(int a,int b) { return a+b; }

Function Call

Calling means executing a function.

int result; result = add(10,20);
Output: 30

Complete Example

#include int add(int,int); int main() { int result; result = add(10,20); printf("%d",result); return 0; } int add(int a,int b) { return a+b; }

Parameters and Arguments

Parameters receive values. Arguments send values.

int add(int a,int b)

a and b are Parameters.

add(10,20);

10 and 20 are Arguments.

Memory Visualization

add(10,20);

Argument 10 goes into parameter a

Argument 20 goes into parameter b

Function returns 30

Functions With No Parameters

void greet() { printf("Welcome"); }

Calling:

greet();

Functions With Return Values

int square(int n) { return n*n; }
square(5) Output = 25

void Functions

void means the function returns nothing.

void hello() { printf("Hello"); }

Local Variables

A Local Variable is declared inside a function. It can only be used within that function.

void test() { int x = 10; printf("%d",x); }
💡 Local variables are destroyed when the function ends.

Global Variables

A Global Variable is declared outside all functions. It can be accessed by every function.

#include int x = 100; void show() { printf("%d",x); }
⚠ Avoid excessive use of global variables because they can make programs difficult to manage.

Call By Value

In C, values are copied and sent to the function. Original variables remain unchanged.

#include void change(int x) { x = 100; } int main() { int a = 10; change(a); printf("%d",a); return 0; }
Output: 10

The original value did not change because only a copy was passed.

🚀 Recursion

Recursion is a technique where a function calls itself.

int factorial(int n) { if(n==1) return 1; return n * factorial(n-1); }
factorial(5) Output: 120

How Recursion Works?

factorial(5) 5 × factorial(4) 5 × 4 × factorial(3) 5 × 4 × 3 × factorial(2) 5 × 4 × 3 × 2 × factorial(1) 5 × 4 × 3 × 2 × 1 120

Real World Example

float calculateCGPA(float totalPoints,int creditHours) { return totalPoints / creditHours; }

Functions help divide large programs into smaller reusable blocks.

Common Mistakes

❌ Forgetting to call the function.
❌ Returning a value from a void function.
❌ Using wrong return type.
❌ Missing semicolon after function prototype.
