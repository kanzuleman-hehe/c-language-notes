What is an Array?

An Array is a collection of elements of the same data type stored in contiguous memory locations. Instead of creating multiple variables, we can store many values in a single array.

💡 Arrays help manage large amounts of data efficiently.

Why Do We Need Arrays?

Without Array:

int m1,m2,m3,m4,m5;

With Array:

int marks[5];

Less code, easier management.

Array Declaration

data_type array_name[size];

Example:

int marks[5];

Array Initialization

int marks[5]={10,20,30,40,50};

The array contains five values.

Accessing Array Elements

printf("%d",marks[0]);
Output: 10

Arrays start from index 0.

Memory Representation

Array Traversal

Visiting every element of an array one by one is called Array Traversal.

for(int i=0;i<5;i++) { printf("%d ",marks[i]); }

Output: 150

Linear Search

Linear Search checks each element one by one.

Sorting arranges elements in order.

2D Arrays

A 2D Array stores data in rows and columns.

int matrix[3][3];

3 Rows × 3 Columns

2D Array Initialization

int matrix[2][2] = { {1,2}, {3,4} };

Display 2D Array

🚀 Advanced Knowledge Corner

Arrays occupy contiguous memory locations. Suppose:

int arr[5];

If arr[0] is stored at address 1000:

arr[0] = 1000 arr[1] = 1004 arr[2] = 1008 arr[3] = 1012 arr[4] = 1016

Because an integer usually occupies 4 bytes.
