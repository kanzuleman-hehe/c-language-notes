What is a String?

A String is a collection of characters stored inside an array. Strings are used to store names, sentences and text data.

💡 String = Character Array + Null Character ('\0')

String Declaration

char name[20];

This string can store up to 19 characters plus the null character.

String Initialization

char name[]="KANZUL";

Memory Representation

String:

char name[]="KANZUL";
K
A
N
Z
U
L
\0

Every string ends with a Null Character.

Input String Using scanf()

char name[20]; scanf("%s",name);
⚠ scanf() stops at spaces.

Input String Using fgets()

char name[50]; fgets(name,50,stdin);
✅ fgets() can read spaces.

Output String

printf("%s",name);

String Length - strlen()

strlen() returns the number of characters.

#include char name[]="KANZUL"; printf("%d",strlen(name));
Output: 6

Copy String - strcpy()

char a[20]="Hello"; char b[20]; strcpy(b,a);

Now b contains Hello.

Concatenate Strings - strcat()

char a[50]="Hello "; char b[]="World"; strcat(a,b); printf("%s",a);
Output: Hello World

Compare Strings - strcmp()

strcmp("Ali","Ali");
Output: 0

0 means both strings are equal.

Access Individual Characters

char name[]="KANZUL"; printf("%c",name[0]);
Output: K

String Traversal

Accessing every character one by one is called String Traversal.

char name[]="Muqtasid Ali"; for(int i=0; name[i]!='\0'; i++) { printf("%c\n",name[i]); }

Reverse a String

char str[]="Muqtasid"; int len=strlen(str); for(int i=len-1;i>=0;i--) { printf("%c",str[i]); }
Output: disatquM

Count Spaces

char str[]="Muqtasid Ali"; int spaces=0; for(int i=0;str[i]!='\0';i++) { if(str[i]==' ') { spaces++; } } printf("%d",spaces);
Output: 1

Convert to Uppercase

char str[]="muqtasid"; for(int i=0;str[i]!='\0';i++) { if(str[i]>='a' && str[i]<='z') { str[i]=str[i]-32; } } printf("%s",str);
Output: MUQTASID

Convert to Lowercase

char str[]="MUQTASID"; for(int i=0;str[i]!='\0';i++) { if(str[i]>='A' && str[i]<='Z') { str[i]=str[i]+32; } } printf("%s",str);
Output: muqtasid

🚀 Advanced Knowledge Corner

Strings are actually character arrays stored in memory.

char name[]="Muqtasid";

Memory Layout:

M u q t a s i d \0

The null character marks the end of the string.

Common Mistakes

❌ Forgetting the null character.
❌ Using scanf() for full names.
❌ Accessing characters outside string length.
❌ Using == instead of strcmp().
