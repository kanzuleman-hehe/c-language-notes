Chapter 7 - Functions in C C Arrays MCQs
C Arrays MCQs

1. What is an array in C?

A) Collection of similar data types
B) Collection of functions
C) Single variable
D) Operator

Answer: A) Collection of similar data types

2. Array index in C starts from?

A) 1
B) 0
C) -1
D) 2

Answer: B) 0

3. Which symbol is used to access array elements?

A) ()
B) []
C) {}
D) //

Answer: B) []

4. Correct declaration of integer array?

A) int arr;
B) int arr[10];
C) array int;
D) int[10] arr;

Answer: B) int arr[10];

5. First element of array arr is?

A) arr[0]
B) arr[1]
C) arr[-1]
D) arr[2]

Answer: A) arr[0]

6. A two dimensional array is also called?

A) Matrix
B) Function
C) Pointer
D) Structure

Answer: A) Matrix

7. Which loop is commonly used to access array elements?

A) for loop
B) switch
C) if
D) goto

Answer: A) for loop

8. Size of array is fixed at?

A) Runtime
B) Compile time
C) Execution end
D) Never fixed

Answer: B) Compile time

9. String in C is stored as an array of?

A) Integers
B) Characters
C) Floats
D) Doubles

Answer: B) Characters

10. Which character marks the end of a string?

A) \n
B) \0
C) \t
D) EOF

Answer: B) \0
