C Language Basics MCQs

1. C language was developed by?

A) James Gosling
B) Dennis Ritchie
C) Bjarne Stroustrup
D) Guido van Rossum

Answer: B) Dennis Ritchie

2. C language was developed at?

A) Microsoft
B) Google
C) Bell Laboratories
D) IBM

Answer: C) Bell Laboratories

3. C language is a ______ language?

A) Low level
B) Middle level
C) Machine level
D) Assembly level

Answer: B) Middle level

4. Which symbol is used to start a comment in C?

A) //
B) ##
C) %%
D) $$

Answer: A) //

5. Which function is the starting point of a C program?

A) start()
B) main()
C) begin()
D) run()

Answer: B) main()

6. Which header file is required for printf()?

A) stdio.h
B) math.h
C) string.h
D) time.h

Answer: A) stdio.h

7. C program file extension is?

A) .cpp
B) .java
C) .c
D) .html

Answer: C) .c

8. Which keyword is used to return a value from function?

A) back
B) return
C) send
D) output

Answer: B) return
