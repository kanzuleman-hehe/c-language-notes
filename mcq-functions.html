Chapter 7 - Functions in C C Functions MCQs
C Functions MCQs

1. What is a function in C?

A) A block of code that performs a task
B) A variable
C) An operator
D) A data type

Answer: A) A block of code that performs a task

2. Which function is the starting point of C program?

A) start()
B) main()
C) run()
D) begin()

Answer: B) main()

3. Which keyword is used to return value from a function?

A) send
B) return
C) back
D) output

Answer: B) return

4. A function declaration is also called?

A) Function prototype
B) Function loop
C) Function body
D) Function call

Answer: A) Function prototype

5. Values passed to a function are called?

A) Arguments
B) Variables
C) Operators
D) Constants

Answer: A) Arguments

6. Variables declared inside a function are called?

A) Global variables
B) Local variables
C) Static variables
D) External variables

Answer: B) Local variables

7. A function calling itself is called?

A) Looping
B) Recursion
C) Iteration
D) Nesting

Answer: B) Recursion

8. Which function does not return any value?

A) int
B) float
C) void
D) char

Answer: C) void

9. How many values can a C function return directly?

A) Unlimited
B) One
C) Two
D) None always

Answer: B) One

10. Which type of function is created by programmer?

A) Library function
B) User-defined function
C) System function
D) Header function

Answer: B) User-defined function
