Chapter 7 - Functions in C C If Else MCQs
C If Else & Switch MCQs

1. Which statement is used for decision making in C?

A) if
B) print
C) scan
D) loop

Answer: A) if

2. Which keyword is used when if condition is false?

A) otherwise
B) else
C) false
D) default

Answer: B) else

3. Which symbol is used to check a condition?

A) :
B) ;
C) Relational operators
D) #

Answer: C) Relational operators

4. Which statement is used for multiple choices?

A) if
B) switch
C) loop
D) goto

Answer: B) switch

5. In switch statement, which keyword is used to stop execution?

A) stop
B) exit
C) break
D) end

Answer: C) break

6. Which keyword runs when no case matches in switch?

A) else
B) default
C) otherwise
D) none

Answer: B) default

7. An if statement inside another if statement is called?

A) Simple if
B) Nested if
C) Switch if
D) Loop if

Answer: B) Nested if

8. Which statement executes one block from many options?

A) switch
B) while
C) for
D) continue

Answer: A) switch

9. Condition in if statement returns?

A) Character
B) Boolean value
C) String
D) Array

Answer: B) Boolean value

10. Which operator is commonly used with conditions?

A) Relational operator
B) Assignment operator
C) Arithmetic operator
D) Increment operator

Answer: A) Relational operator
