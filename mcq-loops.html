C Loops MCQs

1. Which loop is used when number of iterations is known?

A) while loop
B) do while loop
C) for loop
D) switch

Answer: C) for loop

2. Which loop executes at least one time?

A) for loop
B) while loop
C) do while loop
D) nested loop

Answer: C) do while loop

3. Which keyword is used to exit from a loop?

A) stop
B) break
C) exitloop
D) end

Answer: B) break

4. Which keyword skips the current iteration?

A) skip
B) continue
C) pass
D) next

Answer: B) continue

5. Which loop checks condition before execution?

A) while loop
B) do while loop
C) switch
D) function

Answer: A) while loop

6. The three parts of for loop are initialization, condition and?

A) output
B) increment/decrement
C) input
D) return

Answer: B) increment/decrement

7. A loop inside another loop is called?

A) Infinite loop
B) Nested loop
C) Double loop
D) Reverse loop

Answer: B) Nested loop

8. What happens if loop condition never becomes false?

A) Program stops
B) Infinite loop occurs
C) Error occurs
D) Nothing

Answer: B) Infinite loop occurs

9. Which loop is also called exit controlled loop?

A) for
B) while
C) do while
D) nested

Answer: C) do while

10. Which statement can create an empty loop body?

A) for(;;)
B) if()
C) switch()
D) function()

Answer: A) for(;;)
