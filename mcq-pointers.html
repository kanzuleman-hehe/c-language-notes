Chapter 7 - Functions in C C Pointers MCQs
C Pointers MCQs

1. What is a pointer in C?

A) A variable that stores address of another variable
B) A normal variable
C) A function
D) A loop

Answer: A) A variable that stores address of another variable

2. Which symbol is used to declare a pointer?

A) &
B) *
C) #
D) @

Answer: B) *

3. Which operator gives the address of a variable?

A) *
B) &
C) %
D) #

Answer: B) &

4. Which operator is used to access value stored at address?

A) &
B) *
C) +
D) =

Answer: B) *

5. A pointer stores?

A) Value only
B) Address of variable
C) Data type
D) Function name

Answer: B) Address of variable

6. What is a NULL pointer?

A) Pointer with no valid address
B) Pointer with integer value
C) Pointer with string
D) Normal variable

Answer: A) Pointer with no valid address

7. Which data type stores address of integer variable?

A) int
B) int *
C) float
D) char

Answer: B) int *

8. Pointers are mainly used for?

A) Memory management
B) Printing text
C) Creating loops
D) Sorting only

Answer: A) Memory management

9. Which function is used to allocate dynamic memory?

A) printf()
B) malloc()
C) scanf()
D) main()

Answer: B) malloc()

10. Pointer size depends on?

A) Variable value
B) Computer architecture
C) Variable name
D) Data stored

Answer: B) Computer architecture
