C Variables & Data Types MCQs

1. Which keyword is used to declare an integer variable in C?

A) number
B) int
C) integer
D) var

Answer: B) int

2. Which data type is used to store a single character?

A) char
B) string
C) text
D) character

Answer: A) char

3. Size of char data type in C is usually?

A) 1 byte
B) 2 bytes
C) 4 bytes
D) 8 bytes

Answer: A) 1 byte

4. Which data type stores decimal numbers?

A) int
B) char
C) float
D) void

Answer: C) float

5. Which keyword is used to make a variable constant?

A) fixed
B) constant
C) const
D) static

Answer: C) const

6. Which symbol is used for address of a variable?

A) *
B) &
C) #
D) @

Answer: B) &

7. Which data type has no value?

A) int
B) float
C) void
D) char

Answer: C) void

8. Which format specifier is used for integer?

A) %f
B) %c
C) %d
D) %s

Answer: C) %d
